# Copilot Design Instructions

When designing and building web components, pages, or applications, generate creative, polished code that avoids generic AI aesthetics. The goal is to create distinctive frontends that surprise and delight users.

## Frontend Design Principles

### Typography

Choose fonts that are beautiful, unique, and interesting. Avoid overused generic fonts like Arial and Inter—opt instead for distinctive choices that elevate the frontend's aesthetics and create visual interest.

### Color & Theme

- Commit to a cohesive aesthetic throughout the design
- Use CSS variables for consistency across components
- Dominant colors paired with sharp accents outperform timid, evenly-distributed palettes
- Draw inspiration from IDE themes and cultural aesthetics
- Create visual hierarchy through deliberate color choices

### Motion & Animation

- Use animations to enhance user experience and highlight key moments
- Prioritize CSS-only solutions for HTML elements
- Leverage the Motion library for React components when available
- Focus on high-impact moments: one well-orchestrated page load with staggered reveals (using animation-delay) creates more delight than scattered, disconnected micro-interactions
- Let motion guide the user's attention and reinforce the design narrative

### Backgrounds & Atmosphere

- Create atmosphere and depth rather than defaulting to flat, solid colors
- Layer CSS gradients strategically to add dimensionality
- Use geometric patterns that support the overall aesthetic
- Add contextual effects that reinforce the design theme
- Backgrounds should enhance, not distract from, the primary content

## Avoid Generic AI Aesthetics ("AI Slop")

Do not default to these patterns:

- **Overused font families**: Inter, Roboto, Arial, system fonts
- **Clichéd color schemes**: Purple gradients on white backgrounds, overly safe palettes
- **Predictable layouts**: Standard grid patterns with no character or personality
- **Cookie-cutter components**: Design that could apply to any project

Instead, think critically about the context and make unexpected choices that feel genuinely designed for this specific application.

## Implementation Guidance

Match implementation complexity to your aesthetic vision:

- **Maximalist designs** demand elaborate code with extensive animations, layered effects, and intricate details
- **Minimalist or refined designs** require restraint, precision, and careful attention to spacing, typography, kerning, and subtle interactions
- **Elegance** comes from executing your vision consistently and thoroughly

## Creative Vision First

You're capable of extraordinary creative work. Don't hold back—show what can truly be created when:

- Thinking outside conventional design patterns
- Committing fully to a distinctive, context-aware vision
- Pushing boundaries while maintaining usability
- Balancing aesthetic innovation with user experience

Make choices that make people say "wow, that's unique." Design for this bingo game should feel fresh and engaging, not like a template.
