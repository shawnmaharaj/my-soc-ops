# Soc Ops

![Soc Ops screenshot](public/screenshot.png)

A playful, workshop-ready social bingo app that turns introductions into lightning-fast icebreakers.

- 🎉 For team mixers, in-person workshops, and social operations training
- 🧩 Open source React + Vite codebase made for live demos
- 🤖 Includes a Quiz Master AI style extension flow (homegrown in the workshop)

---

## 🚀 Why this project

Soc Ops is built to make collaborative onboarding fun:

- Run a **real-time mobile-friendly bingo board** with question-based prompts
- Customize people-finding cards in `src/data/questions.ts`
- Track, highlight, and celebrate `BINGO` states with accessible UI
- Learn modern front-end engineering through step-by-step lab guides

---

## 🎯 Try it now

1. Clone the repository
2. Run:

```bash
npm install
npm run dev
```

3. Open `http://localhost:5173`
4. Invite colleagues, read a card, connect, and complete 5 in a row

> Tip: Use the workshop guides for “Quiz Master” and “Multi-Agent” extensions to level this from game to demo platform.

## 🛠️ Quick dev workflow


### Publish

GitHub Pages deploys automatically from `main` on push.
